
# BACKEND GOLANG (ECHO) 📝  
## Technology
- Golang (echo)
- Gorm
- Mysql
- Env
- CORS

## Get Started 🚀  
Install all package using go
```bash
go mod tidy
go mod download
go get
```

## Usage 🔥  
local
```bash
go run main.go
```
running on port localhost:8000
